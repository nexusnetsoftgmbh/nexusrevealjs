What is ChatGPT?

  • ChatGPT is a large language model created by OpenAI
  • Based on the GPT-3 text prediction algorithm
  • ChatGPT-3 was fed 45 TB of data
  • Latest dataset is from 2021
  • Machine learning is used to teach it
  • You can ask it questions
  • ChatGPT-3 is basically a chat UI with build in filters (e.g bombs, politics and drugs), which gives the users easy access to commands (prompts)
  • Under the hood it uses the GPT-3
  • In short GPT-3 just predicts which word comes possibly next. You can compare it to your smartphone
  • this data includes books, articles, websites and manually added information
  • Its important to remember, that GPTs knowledge is only as good, as its data, the ML algorithm and the prompt
  • Flat earth example, ML algorithm basically up- and downvotes provided answers to teach the model
  • Can be slightly compared to teach a trick to a dog. Executed wanted command = treat
  • Everything after 2021 is currently not fully known

The importance of clear instructions

  • I think many of you already saw that this video has many connections to our work
  • If a developer, designer or even a craftsman gets no clear instructions it can end in an unexpected outcome
  • To increase the quality of ChatGPTs answers it's recommended to follow this simple three rules

The Three Rules

  • Be as specific as possible
  • Give it as much context information as possible
  • Don’t blindly trust ChatGPT answers - Verify it!
  • Like seen in the video, clear instructions are very important
  • You get more precise answers to your specific question with added context information
  • My computer is overheating - how do I fix that? Add more information to it like My Desktop PC is overheating when im playing games
  • Its very important to verify its output, because it can hallucinate
  • a solution may sound plausible, but can still be wrong if someone if domain knowledge checks it

How can it Support us with our Problems?

  • Creating Unit- or Integration tests for your code
  • Optimizing the text of an article to have better SEO results
  • Give you a guide to create a complex project specific excel formular
  • Providing a list of pros & cons regarding a new technology
  • Just copy and paste your code into chatgpt and say it should write phpunit tests
  • Give it instructions to include keywords to your target audience
  • Can also help with a tutorial for any other application
  • For example if you want to evaluate if the new technology would match your requirements
